# Express-js-Server-Mongodb-integration# Express + MongoDB Server

This project is a simple **Express.js server** integrated with **MongoDB using Mongoose**.  
It demonstrates basic routing, JSON responses, and database connectivity.

---

## 🚀 Features
- Basic Express server setup
- Routes for:
  - `/` → Home Page
  - `/data` → Sample text response
  - `/joke` → Returns a list of jokes in JSON format
- MongoDB connection using Mongoose
- Console logs for server and database status
